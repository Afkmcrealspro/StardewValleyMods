## 1.2.0
```
Sat, 05 Aug 2017 09:38:37 -0400
Added some features.
Downloads:
    1.2.0
Full Changelog:
    Support for glowing.
    A "random" behavior type, which uses patrol points but selects them randomly.
    An "offset" patrol point type, which is offset from the current position instead of start.
    A "startrandom" patrol point type, which chooses a random point within a square rectangle (width/height specified as X/Y of the point) based from the starting position to go to.
    A "random" patrol point type, which is the same as "startrandom" except relative to the current position instead of the starting position.
    A "wait" patrol point type, which does nothing but wait another cycle.
```
